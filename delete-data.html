Exclusão de Dados – FitSwap

O FitSwap permite que os usuários excluam dados específicos sem a necessidade de excluir a conta.

Como excluir dados

  1. Abra o aplicativo FitSwap.
  2. Localize o conteúdo publicado (foto ou vídeo).
  3. Toque nos três pontos do conteúdo.
  4. Selecione a opção "Excluir".

Tipos de dados que podem ser excluídos

  • Fotos
  • Vídeos
  • Conteúdos publicados pelo usuário

A exclusão é imediata e não afeta a conta do usuário.
